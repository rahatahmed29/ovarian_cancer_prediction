# Predicting Ovarian Cancer Using WOA-Based XGBoost
This repository contains the complete implementation of our undergraduate thesis on
ovarian cancer prediction using Whale Optimization Algorithm based XGBoost classifier

## Dataset
- Source: Mendeley Data
- Ethical approval: Yes
- Link: https://data.mendeley.com/datasets/th7fztbrv9/11

## Methodology
1. Data acquisition
2. Data preprocessing
3. Baseline model evaluation (RF, CatBoost, XGBoost)
4. Stable feature selection using WOA
5. Hyperparameter tuning using WOA
6. Final performance analysis 
## Models Used
- Random Forest
- CatBoost
- XGBoost (final model)
